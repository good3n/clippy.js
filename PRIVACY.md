# Privacy Policy

**Clippy Agent Chrome Extension**
Last updated: April 1, 2026

## What this extension does

Clippy Agent displays an animated character on webpages in your browser. It can optionally connect to an AI backend you configure yourself.

## Data we collect

None. Zero. We do not collect, store, transmit, or sell any user data.

## Data stored on your device

The extension stores the following in your browser's local storage (chrome.storage.sync):

- Your selected agent character (e.g. "Clippy", "Merlin")
- Whether AI features are enabled (true/false)
- Your AI gateway URL and authentication token, if you choose to configure one

This data never leaves your browser unless you enable AI features and configure your own gateway.

## AI features

AI features are off by default. If you enable them, the extension sends page context (page title, URL, visible text, selected text) to whatever gateway URL you configure in the settings. This is your own server. We do not operate, control, or have access to your gateway. We never see your queries or responses.

We do not provide, require, or request any API keys. We do not proxy requests through our servers. We have no servers.

## Remote resources

The extension loads agent sprite sheets and animation data from unpkg.com (the npm package registry CDN). These are static image and JSON files. No executable code is loaded remotely. unpkg.com may log standard HTTP request data (IP address, user agent) per their own privacy policy.

## Analytics and tracking

None. No analytics. No telemetry. No tracking pixels. No fingerprinting. No cookies.

## Third-party services

We do not integrate with any third-party analytics, advertising, or data collection services.

## Permissions

The extension requests these Chrome permissions:

- **All URLs (host permissions):** Required to display the animated character on every webpage.
- **Context menus:** Adds right-click options for switching agents and asking Clippy about selected text.
- **Storage:** Saves your preferences locally in the browser.

## Children's privacy

We do not knowingly collect any information from anyone, including children under 13.

## Changes to this policy

If this policy changes, the updated version will be posted here with a new "last updated" date. Since we collect no data, there's not much to change.
