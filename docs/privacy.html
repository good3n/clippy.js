Privacy Policy

Last updated: April 1, 2026

What this extension does

Clippy Agent displays an animated character on webpages in your browser. It can optionally connect to an AI backend that you configure yourself.

Data we collect

None. We do not collect, store, transmit, or sell any user data.

Data stored on your device

The extension stores the following in your browser's local storage:

  • Your selected agent character (e.g. "Clippy", "Merlin")
  • Whether AI features are enabled
  • Your AI gateway URL and token, if you choose to configure one

This data never leaves your browser unless you enable AI features and configure your own gateway.

AI features

AI features are off by default. If you enable them, the extension sends page context (title, URL, visible text) to whatever gateway URL you configure. That's your server. We don't operate it, we don't have access to it, and we never see your queries or responses.

We don't provide, require, or request any API keys. We don't proxy requests through our servers. We have no servers.

Remote resources

The extension loads sprite sheets and animation data from unpkg.com (the npm registry CDN). These are static image and JSON files. No executable code is loaded remotely. unpkg.com may log standard HTTP request data per their own privacy policy.

Analytics and tracking

None. No analytics, no telemetry, no tracking pixels, no fingerprinting, no cookies.

Third-party services

We do not use any third-party analytics, advertising, or data collection services.

Permissions

  • All URLs: Required to display the animated character on every webpage.
  • Context menus: Right-click options for switching agents and asking about selected text.
  • Storage: Saves your preferences locally in the browser.

Children's privacy

We don't knowingly collect any information from anyone, including children under 13.

Changes

If this policy changes, the updated version will be posted here with a new date. Since we collect no data, there's not much to change.
